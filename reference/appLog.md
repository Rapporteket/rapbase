# App log test dataset.

A dataset containing test entries for the application log.

## Usage

``` r
appLog
```

## Format

A data frame with 20 rows and 8 variables:

- id:

  character id

- time:

  character timestamp

- user:

  user name

- name:

  user full name

- group:

  users group/registry

- role:

  users role

- resh_id:

  users organization

- message:

  log message
