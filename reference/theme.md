# Theme of the app.

This is a wrapper around bslib::bs_theme

## Usage

``` r
theme(theme = "flatly", version = 3)
```

## Arguments

- theme:

  Name of the theme. See bslib::bootswatch_themes() for available
  themes.

- version:

  Version of bootstrap to use (3, 4 or 5).

## Value

a Bootstrap theme object
